# Sprites / Personajes

Coloca aquí las imágenes de personajes en formato PNG con fondo transparente.

**Resolución recomendada:** 800x1200px (proporción vertical)

## Lista de sprites necesarios:

### Gabriel (Caballero)
- `gabriel_neutral.png` - Expresión neutral
- `gabriel_determined.png` - Determinado
- `gabriel_alert.png` - Alerta/Defensivo
- `gabriel_happy.png` - Feliz
- `gabriel_sad.png` - Triste/Vulnerable

### Allison (Princesa)
- `allison_neutral.png` - Expresión neutral
- `allison_surprised.png` - Sorprendida
- `allison_determined.png` - Determinada
- `allison_happy.png` - Feliz

### Otros Personajes
- `skeleton_guard.png` - Guardia esqueleto con armadura
- `dragon_awake.png` - Dragón negro imponente

## Nota:
Si no tienes los sprites, el juego los ocultará automáticamente. El juego es completamente funcional sin sprites.
