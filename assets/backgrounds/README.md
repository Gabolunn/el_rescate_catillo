# Fondos / Backgrounds

Coloca aquí las imágenes de fondo en formato JPG o PNG.

**Resolución recomendada:** 1920x1080px (16:9)

## Lista de fondos necesarios:

- `bg_castle_entrance.jpg` - Entrada del castillo de noche
- `bg_castle_interior.jpg` - Pasillo interior del castillo
- `bg_dungeon.jpg` - Calabozos húmedos
- `bg_dungeon_cell.jpg` - Interior de una celda
- `bg_tower_stairs.jpg` - Escaleras de caracol
- `bg_tower_hall.jpg` - Pasillo de la torre
- `bg_dragon_chamber.jpg` - Cámara del dragón con tesoros
- `bg_tower_top.jpg` - Cima de la torre
- `bg_allison_chamber.jpg` - Habitación de Allison
- `bg_secret_passage.jpg` - Pasadizo secreto
- `bg_garden.jpg` - Jardín secreto nocturno
- `bg_forest_exit.jpg` - Salida del castillo al bosque
- `bg_kingdom.jpg` - Vista del reino
- `bg_library.jpg` - Gran biblioteca
- `bg_sunset.jpg` - Atardecer romántico
- `bg_throne_room.jpg` - Sala del trono
- `bg_black.jpg` - Fondo negro puro (o dejar que CSS lo maneje)

## Nota:
Si no tienes los assets, el juego usará gradientes de color como placeholder.
